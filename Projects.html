CPU & GPU Rasterizer

After having made a Software Rasterizer, the next step was to make a hardware rasterizer using DirectX11. Using the Effects framework, my final project for Graphics Programming 1 was to make a merged renderer that can easily switch between the right-handed software rasterizer and the left-handed hardware rasterizer. The goal was to have a seamless transition at runtime between the two different rasterizers, atemting to make them look as similar as possible.

CPU Ray Tracer

This is the first software ray tracer I wrote in C++. I implemented a dynamic scene system to easily set up scenes ans switch between them. The ray tracer can handle simple shapes such as spheres and planes, it can also render triangle meshes such as the 'Stanford Bunny'. Point lights and directional light have been implemented. To look around the scene I added multiple controlls, so the camera can be easily be controlled with just the keyboard, or if prefered, both keyboard and mouse. for materials I have added several BRDFs such as Phong, Lambert and Cook-Torrance. Some aditional functionallity I added is the ability to toggle shadows on/off, cycle cullmode, cycle rendering mode, and toggle some lights on/off.

Research Project

Comparing sphere generations algorithms applied to procedural Planets

From the 28th of September 2021 I have this as my graduation research project. With this I would like to investigate the different pros and cons with multiple sphere generation algorithms like the UV Sphere, Quad Sphere, IcoSphere and the Fibonacci Sphere. I hope to compare the difference in Vertex distribution, vertex density, the difference in stored data, how much controll we have when subdividing, which lends it self best to utilize the UV space most and which has the best poligonal uniformity. For this there will be made a interractive Sphere generation software where we will be able to apply textures, noice-maps and adjust other parameters to generate a planet like object. The target object to replicate will be both the Moon and the Earth.
